保険料 が 免除 に なる と 、 生命保険 料 が 無い から 、 控除 す べき 出費 が ない ので 、 該当 し なく なり ます 。 （ 死亡 前 に ） 保険料 を 支払っ て いれ ば 、 生命保険 控除 用 の 証明書 が 来 ます ので 対象 です 。 
整体 で の 労災 使用 に は 、 まず 整形 外科 なり の 「 医師 」 の 診断 が 必要 です 。 整体 の 先生 は 医師 で は あり ませ ん ので 。 ①「 医師 」 に 病名 を 診断 し て もらう ② 会社 か 個人 か で 労災 申請 を する 。 ③ 労災認定 が 下りる ④「 医師 」 に 整体 で の 治療 の 必要性 の 診断 を し て もらう この 手順 で 、 初めて 整体 で の 治療 が 労災 扱い と なり ます 。 ここ で 一 点 問題 な の は 、 腰痛 は 非常 に 労災認定 さ れ にくい という 点 です 。 なぜなら 「 労働 上 が 原因 」 で なく て も 、 普通 の 生活 や 家事 で も なり ます し 、 専業主婦 や 学生 で も なり ます 。 それ を 労働 上 の 事 が 原因 で 起き まし た 、 と 証明 する の は かなり 難しい の です 。 証明 する の は あなた です し ... 普通 、 証拠 なんて ありません から 。 なので 労災認定 基準 も かなり 厳しく なっ て い ます 。 極端 に 重い もの を 無理 な 姿勢 で 長時間 扱っ た 、 腰痛 に なる こと が 予測 さ れ た ので 対策 を 会社 に 求め た が 無視 さ れ た 、 など の 要因 が 無い と 、 難しい と 思っ て ください 。 
自治体 によって 対応 が 異なり ます 。 基本的 に は 加入 に は 健康保険 の 資格 喪失 証明 が 必要 です 。 です が 自治体 によって は 離職票 や 退職 証明書 で も 代用 可能 と し て い ます 。 また 、 担当者 が 会社 に 電話 確認 を し て それ で よし と する こと も ある らしい です 。 （ どこ でも できる と は 思え ませ ん が ） まずは 役所 に 確認 し て もらう しか あり ませ ん 。 健康保険 の 資格 喪失 証明 しか 受け付け ない という こと で あれ ば 手 に 入れる しか あり ませ ん 。 前 の 会社 に 頼め ない として も 、 加入 し て い た 健康保険 に 発行 依頼 でき ます 。
